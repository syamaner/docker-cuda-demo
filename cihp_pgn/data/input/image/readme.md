The images in this directory are generated using StyleGAN-Human demo using Hugging Face space demos as below:

Demo:
https://huggingface.co/spaces/hysts/StyleGAN-Human-Interpolation

Repository:
https://github.com/stylegan-human/StyleGAN-Human

- Truncation psi 1: 0.7
- Seed 2: 1
- Truncation psi 2: 0.7
- Number of intermediate Frames: 7
